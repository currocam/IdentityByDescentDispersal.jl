IdentityByDescentDispersal.jl

IdentityByDescentDispersal.jl is a Julia package for estimating effective population densities and dispersal rates from observed spatial patterns of IBD shared blocks. It provides the building blocks for performing likelihood-based inference and integrates seamlessly with other statistical libraries in the Julia ecosystem, such as Turing.jl.

Installation

You can install this package by running:

import Pkg
Pkg.add("IdentityByDescentDispersal")
